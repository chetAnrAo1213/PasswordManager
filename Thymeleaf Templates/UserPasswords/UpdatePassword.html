Updating Process

How Does update process work?

Updating the password involves specifying the Password Id which user would'nt know unless the user visits the view page. The Id is being generated by the system and based on the Id the entire password credentials will be changed and the old credentials will be deleted permanently.

How does Password Id generated?

For each and every user's password the system will generate an unique value. These values are generated in an sequence order results in no duplication of the Id. This Id helps the user to identify a particular password when you visit the View Page.
