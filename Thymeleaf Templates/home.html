Password Manager

How does this app works?

Password Manager is a tool to store your passwords. User can start storing them in by navigating to the Passwords option in the Nav Bar.

Will passwords be safe?

We use AES Encryption of 128-Rounds & the encrypted
passwords will be stored in our system. The encrypted passwords can be decrypted in our decryption page.

Features of this app?

Users will get an AI-Model ChatBot and Quick Note and interactive external API's. Also personlized reports of the users to track their usage
